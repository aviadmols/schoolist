# ✅ רשימת בדיקה לעלייה לשרת

רשימת בדיקה מהירה לפני ואחרי העלייה לשרת.

## 📋 לפני העלייה

- [ ] הפרויקט עובד מקומית
- [ ] בדקתי את כל הפונקציונליות
- [ ] יש לי פרטי מסד נתונים (host, DB name, user, password)
- [ ] יש לי כתובת URL של האתר
- [ ] יש לי פרטי SMTP (אם נדרש)
- [ ] יש לי OpenAI API Key (אם נדרש)
- [ ] מחקתי קבצים לא נחוצים (test_*.php, admin_helper.php, וכו')

## 📤 העלאת קבצים

- [ ] העליתי את כל הקבצים לשרת
- [ ] `index.php` נמצא ב-root
- [ ] `.htaccess` נמצא ב-root
- [ ] התיקיות `app/`, `config/`, `public/`, וכו' עלו

## 🔐 הרשאות

- [ ] `chmod 755 config/`
- [ ] `chmod 755 public/uploads/`
- [ ] `chmod 755 logs/`
- [ ] `chmod 755 storage/`
- [ ] `chmod 755 storage/ratelimit/`

## 🗄️ מסד נתונים

- [ ] יצרתי מסד נתונים חדש
- [ ] יצרתי משתמש למסד הנתונים
- [ ] הקציתי הרשאות למשתמש
- [ ] ייבאתי את `database/schema.sql`
- [ ] בדקתי שכל הטבלאות נוצרו

## ⚙️ Setup Wizard

- [ ] פתחתי `/setup` בדפדפן
- [ ] השלמתי שלב 1 (בדיקת דרישות) ✅
- [ ] השלמתי שלב 2 (מסד נתונים) ✅
- [ ] השלמתי שלב 3 (Base URL) ✅
- [ ] השלמתי שלב 4 (Email/SMTP) ✅
- [ ] השלמתי שלב 5 (סיום + Admin) ✅
- [ ] `config/config.php` נוצר
- [ ] `setup.lock` נוצר

## ✅ בדיקות אחרי התקנה

- [ ] `/setup` מציג "המערכת כבר מותקנת"
- [ ] `/login` עובד
- [ ] התחברתי בהצלחה עם האימייל שלי
- [ ] `/admin` נגיש
- [ ] `/p/{id}` עובד (אחרי יצירת דף)
- [ ] העלאת קבצים עובדת
- [ ] אימיילים נשלחים (אם הוגדר SMTP)

## 🔒 אבטחה

- [ ] האתר רץ על HTTPS
- [ ] `config/config.php` מוגן (לא נגיש דרך דפדפן)
- [ ] `setup.lock` קיים
- [ ] מחקתי קבצים לא נחוצים (`admin_helper.php`, `test_*.php`)
- [ ] בדקתי את `logs/app.log` - אין שגיאות קריטיות
